PVSio-webRapid prototyping interactive systems using formal methods.

PVSio-web is a graphical toolkit that transforms the animation capabilities of the standard PVS theorem proving system with a sophisticated front-end allowing interactive (human-computer) systems modelling and prototyping. Using PVSio-web, one can generate and evaluate realistic interactive prototypes based on formal models. It has been successfully used for analysing commercial medical devices, as well as to create training material for device developers and end users (see featured videos and publications for details).
